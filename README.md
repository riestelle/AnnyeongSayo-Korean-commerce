# Annyeong'Sayo Korean Mini Store
### You may visit the site here: [Annyeong'Sayo](https://annyeong-sayo.infinityfreeapp.com/index.php?i=1)

An integrated commerce platform that combines:
- Online Shop (customer-facing storefront)
- Admin Dashboard (management and analytics)
- Cashier UI (point-of-sale interface)

## Features
- Product browsing and purchase flow
- Secure checkout and order management
- Admin tools for inventory, users, and reports
- Cashier interface for in-person transactions
- Responsive design for desktop and mobile

## Tech Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: PHP / MySQL
- Deployment: InfinityFree

## Getting Started
1. Clone the repository:
```
git clone https://github.com/riestelle/AnnyeongSayo-Korean-commerce.git
```
3. Set up your local PHP/MySQL environment.
4. Import the provided database schema (`korean_store`).
5. Run the project locally or deploy to your preferred hosting.

## Project Scope
This is more than a simple demo shop — it’s a **full commerce system** showcasing:
- Online shopping experience
- Administrative control panel
- Cashier interface for physical transactions
